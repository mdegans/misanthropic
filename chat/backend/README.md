# Chat Backend

This is the backend for the chat demo. It is built using Shuttle.

## Running the Backend

1. Install the [`shuttle` tool](https://www.shuttle.dev/).

Run

```
shuttle run -p 8079
```

See the [Frontend README](../frontend/README.md) for instructions on how to run the frontend. Both the frontend and backend need to be running for the app to work. Startup order does not matter.
